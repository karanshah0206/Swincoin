# Swincoin
* Byzantine Fault Tolerant
* SHA256 Hashes
* Nonce Generation (Complexity: 5)
* RSA Pub/Secret Keys
* Multithreaded
* Decentralized (Nodes Run As Websockets On Localhost)
* Genesis Reward: 50000 Swincoins
* Miner Reward: 15 Swincoins
